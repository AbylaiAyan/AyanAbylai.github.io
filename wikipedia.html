Курсы High School Program

Курсы High School Program — курсы по обучению абитуриентов базовым знаниям программирования, главными целями которого являются:

  • Научить абитуриентов программировать на Java
  • Научить использовать современный подход объектно-ориентированного программирования (ООП)
  • Научить создавать приложения с анимацией, звуками и управлением через клавиатуру
  • Предоставить сертификаты от iCarnegie, которые будут учтены при поступлении в IITU (МУИТ)
  • Предоставить возможность поработать над интересными проектами
  • Предоставить возможность обучиться у почетного профессора с США по международным стандартам
  • Научить создавать web-страницы с играми
  • Багжан Карл

Лаборатории

Облачные вычисления

Лаборатория Облачных вычислений Huawei(Cloud Computing of Innogrid) — 9 августа 2011 года была создана лаборатория открытых систем и облачных вычислений при тесном сотрудничестве китайской компании Huawei, инфокоммуникационного холдинга “Зерде” и Международного Университета Информационных Технологии. Основная цель создания лаборатории сфокусирована на научно-исследовательских работ в сфере “облачные вычисления”, “открытые системы” с использованием технологии Open Source.[4]

Лаборатория Apple

Разработка приложений для мобильных устройств на iOS(iOS Application Development) — курс ознакомит студентов с языком программирования Objective C и разработкой приложений для мобильных устройств на базе iOS. По окончании курса студенты смогут самостоятельно разрабатывать приложения и программы на Objective C для iPhone, iPad и других iOS-устройств, а также работать в среде разработки XCode. Курс включает в себя введение в язык Objective C, применение концепций объектно-ориентированного программирования при разработке на этом языке, изучение парадигмы разработки MVC, работа с различными компонентами в среде разработки Xcode. Перед началом курса студент должен иметь представление об объектно-ориентированном программировании, быть знакомыми с синтаксисом С-подобных языков и иметь базовые навыки использования графических средств разработки, компиляторов и отладчиков.

Разработка приложений и программ для Mac OS X (Mac OS X Application development) — курс научит студентов разрабатывать приложения для операционной системы Mac OS X. По окончании курса у студентов будет опыт работы с Cocoa Framework и Objective C. Требуется предварительное знание языка программирования Objective C. Курс включает в себя обзор платформы и операционной системы Mac OS X, принципов проектирования интерфейсов приложений для персональных компьютеров.

Advanced Apple Development — курс предназначен для углубленного изучения методов и средств разработки приложений на основе технологий Apple. Курс включает в себя более детальное изучение языка Objective C, дополнительных и сторонних библиотек, среды разработки XCode, работы с отладчиком, методов комбинирования различных языков и технологий с программами на Objective C, фреймворков WebKit, ParseKit, Cocoa, RestKit, написания веб-приложений с помощью Objective C и XCode, архитектуры операционных систем Apple Mac OS X, Apple iOS, ядра Darwin, а также аппаратная архитектура устройств AppleiPhone/iPad.

Лаборатория Microsoft

Лаборатория 3D графики и анимации

Лаборатория 3D графики и анимации была открыта при Международном Университете Информационных Технологий в 2013 году. Несмотря на это, лаборатори, успела сотворить несколько качественных продуктов на казахстанский рынок. Основным направлением лаборатории является 3D Mapping Show, а так же создание видео презентаций и различных 3D проектов.
